Top 10 VS Code shortcuts:

1. 'Ctrl + P' -- Quick file open. 
2. 'Ctrl + Shift + P' -- Open the command palette. 
3. 'Ctrl + \' -- Split editor. 
4. 'Ctrl + Tab' -- Change file tabs/panels. 
5. 'Ctrl + /' -- Toggle line comment. 
6. 'Alt + Up' or 'Alt + Down' -- Move line up and down. 
7. 'Ctrl + Enter' -- Insert a new line below. 
8. 'Ctrl + Shift + K' -- Remove line. 
9. 'Ctrl + Shift + O' -- Go to symbol (function/var) in file. 
10. 'Ctrl + +' or 'Ctrl + -' -- Zoom in/out.


Which shortcuts do you think will be the most useful in your daily work?
'Alt + Up' or 'Alt + Down' -- extremely useful tool to move code lines without copying and pasting.
'Ctrl + Tab' -- useful to change between files when working on multiple components
'Ctrl + /' -- toggling comments will instantly help debugging the code

How can keyboard shortcuts improve your productivity?
Keyboard shortcuts improve productivity by minimising the need to use mouse. This saves a lot of time when solving problems. The shortcuts will allow to edit, navigate, debug, and write code much more efficiently.
